# my-customize-conky
 - you need to install conky in your computer 
 - place this conkyrc file into your home .conky folder
 - you can run
 ``` bash
 $ conky -c ~/$HOME/.conky/custom.conkyrc 
 or
 $ bash start.sh
 ```
